## Installation

```
$ apm install night-ui
```

## Screenshot

![Night UI Screenshot](https://i.imgur.com/XF1y0YM.png)

## Setup
1. Go to your Preferences and click on Themes in the sidebar.
2. Click on the `UI Theme` dropdown and change the value to `Night`.
